Los datos se pueden encontrar en la siguiente página: https://www.kaggle.com/c/recognizing-faces-in-the-wild/data

Los datos de entrenamiento y de validación se encuentran en el archivo test-public-faces.zip. El CSV con
las relaciones correspondientes es train_relationships.csv.

Para los datos de test, hemos utilizado el archivo test.zip. El CSV con las relaciones que se quieren
probar es sample_submissions.csv.
